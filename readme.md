 <br>

## Dev

### - Backend:
1. Clone the project.
2. Set up Docker.
3. Run `docker compose up -d`.
4. Enter the project's database and JWT seed in the `.env.template` file.
5. Rename `.env.template` to `.env`.
6. Run `npm install`.
7. Run `npm run start`.

### - Frontend:
1. Clone the project.
2. Run `npm install`.
3. Launch the app with `npm start` or `ng serve -o`.
